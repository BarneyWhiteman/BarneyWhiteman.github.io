Fragment Collapse is a satisfying tile matching roguelike where you defragment your hard drive by removing clusters of files. Discover powerful cartridges to upgrade your computer, boost your files with enhancements, and build massive combos for file-busting mayhem!

Fragment Collapse is currently in development and I'm aiming to release it in around September 2026 (BUT you can WISHLIST it now on Steam!) I'm making the game in Java using Processing as a graphics library, and everything else is done from scratch!
If you'd like to keep up to date with the game, offer feedback, or report a bug, the best place to do that is on my Discord server.

If you're interested in learning more about the development of the game, I've got a few live streams on my YouTube channel that you can watch back.
