The Tile Matching Roguelike! Enhance your pieces, unlock powerful cartridges, and manipulate the grid for tile matching mayhem! Build combos and synergies to be the ever increasing score target!

Fragment Collapse is currently in development and I'm aiming to release it in around September 2026. I'm making the game in Java using Processing as a graphics library, and everything else is done from scratch!
If you'd like to keep up to date with the game, offer feedback, or report a bug, the best place to do that is on my Discord server.

If you're interested in learning more about the development of the game, I've got a few live streams on my YouTube channel that you can watch back.
