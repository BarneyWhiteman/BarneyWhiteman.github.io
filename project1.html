Spikey Bird is my first venture into the world of mobile games. I made this app using the open source Godot Engine and had a blast making it. It is not a very polished product, however it provided a lot of experience about what it takes to create a game from start to finish (minus the updates and maintainence of course!).

While I did have plans for more features in this game, I won't be continuing to devolp it as it was more of an experiment than anything else.
